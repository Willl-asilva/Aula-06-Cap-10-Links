Como funciona o MVC

  • Requisição do Utilizador: O utilizador interage com a View (por exemplo, clicando num botão num website).


  • Controller: O Controller recebe a requisição, processa os dados da interface e decide qual ação tomar.


  • Model: O Controller pode então interagir com o Model, que é responsável pela lógica de negócios e acesso aos dados (como um banco de dados).


  • Resposta e Exibição: O Model devolve os dados ao Controller, que seleciona a View adequada para exibir esses dados. A View, por sua vez, é enviada de volta para o utilizador, completando o ciclo.
